Backend:

1. cd backend
2. npm install
3. node app.js
4. create .env and put the environment variables:
    PRAYER_URL_API=<apiurl>
    #firebase config
    API_KEY=""
    AUTH_DOMAIN=""
    PROJECT_ID=""
    STORAGE_BUCKET=""
    MESSAGING_SENDER_ID=""
    APP_ID=""
    
    PORT=5000
    HOST=localhost
    HOST_URL=http://localhost:5000

5. check swagger docs:
   http://localhost:5000/api-docs/
